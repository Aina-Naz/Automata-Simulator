# Automata Theory Simulator
This is a web-based tool to simulate Deterministic Finite Automata (DFA) and Nondeterministic Finite Automata (NFA).  

# How to Run on Your Laptop
To run this project locally, follow these simple steps:

# Download the Project: 
Click on the green Code button and select Download ZIP, then extract the files on your computer.  

# Setup Folder: 
Create a new folder on your Desktop named "Automata Simulator" and copy the three extracted files (index.html, script.js, and style.css) into this folder.  

# Launch:
Simply double-click the index.html file to run the simulator in your default web browser (Chrome, Edge, or Firefox).  

# Editing:
If you wish to make any changes or edit the code, you can open the entire folder in VS Code (Visual Studio Code).
